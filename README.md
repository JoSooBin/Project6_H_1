# Project6_H_1

캘린더

- 데이트피커를 사용하여 날짜를 설정한다.

-  <예약시작>기능은 크로노미터를 클릭하면 동작, <예약완료> 기능은 아래쪽 연도를 클릭하면 동작.

- 크로노미터를 클릭하기 전에는 라디오버튼, 데이트피커, 타임피커가 안 보이도록 설정,   크로노미터를 클릭하면 라디오 버튼이 나타나게 한다.   그리고 아래쪽 연도를 클릭하면 라디오버튼, 데이트피커, 타임피거가 다시 사라지게 한다.
